Политика конфиденциальности

Последнее обновление: 2 апреля 2026 г.

Настоящая политика конфиденциальности описывает, как приложение NomadPath («мы», «приложение») собирает, использует и защищает ваши данные. Используя приложение, вы соглашаетесь с условиями данной политики.

1. Разработчик

NomadPath — индивидуальный разработчик.
Контакт: nomadpath.ru@gmail.com

2. Какие данные мы собираем

  • Учётная запись: email-адрес и идентификатор пользователя через Firebase Authentication (Google Sign-In или email/пароль).
  • Геолокация: координаты GPS для записи маршрутов и отображения на карте. Сбор происходит только при явном запуске трекинга пользователем, в том числе в фоновом режиме при активном сеансе записи.
  • Данные здоровья: количество шагов, пройденное расстояние, сожжённые калории — через Android Health Connect. Данные используются исключительно для отображения статистики внутри приложения.
  • Голосовой ввод: аудио с микрофона используется для распознавания речи при взаимодействии с ИИ-ассистентом. Аудио не сохраняется и не передаётся третьим лицам.
  • Контент пользователя: квесты, дневниковые записи, заметки, финансовые данные — хранятся в облаке Firebase Firestore и привязаны к вашему аккаунту.
  • Техническая информация: тип устройства, версия ОС, логи ошибок — для диагностики и улучшения приложения.

3. Разрешения приложения

  • INTERNET, ACCESS_NETWORK_STATE — для синхронизации данных с облаком и работы ИИ.
  • RECORD_AUDIO — для голосового ввода в ИИ-ассистент.
  • ACCESS_FINE_LOCATION, ACCESS_COARSE_LOCATION — для трекинга маршрутов на карте.
  • ACCESS_BACKGROUND_LOCATION — для продолжения записи маршрута при свёрнутом приложении.
  • FOREGROUND_SERVICE, FOREGROUND_SERVICE_LOCATION — для работы фонового сервиса трекинга.
  • POST_NOTIFICATIONS — для уведомлений о квестах и событиях.
  • ACTIVITY_RECOGNITION — для подсчёта шагов.
  • Health Connect (READ_STEPS, READ_DISTANCE, READ_ACTIVE_CALORIES_BURNED) — для чтения данных о физической активности из Health Connect.

4. Как мы используем данные

  • Авторизация и идентификация пользователя.
  • Хранение игрового прогресса, квестов и записей дневника.
  • Генерация персонализированных квестов и ответов ИИ-ассистента.
  • Отображение статистики активности и маршрутов.
  • Улучшение приложения на основе агрегированных данных об ошибках.

5. Передача данных третьим лицам

Мы не продаём и не передаём ваши персональные данные третьим лицам. Данные обрабатываются следующими сервисами:

  • Google Firebase (Authentication, Firestore) — хранение данных аккаунта и контента. Политика Google.
  • Google Gemini API — обработка текстовых запросов для ИИ-функций. Запросы не содержат персональных данных. Политика Google.
  • Google Play Billing — обработка внутриигровых покупок. Политика Google Payments.

6. Хранение и защита данных

Данные хранятся на серверах Google Firebase в соответствии со стандартами безопасности Google. Локально на устройстве данные кэшируются в зашифрованном хранилище Hive для работы в офлайн-режиме.

7. Удаление данных

Вы можете удалить свой аккаунт и все связанные данные в любой момент через раздел Профиль → Удалить аккаунт внутри приложения. Все данные на серверах удаляются безвозвратно в течение 30 дней.

8. Данные детей

Приложение не предназначено для детей до 13 лет. Мы не собираем намеренно данные пользователей младше 13 лет.

9. Изменения политики

О существенных изменениях политики конфиденциальности мы уведомим через уведомление в приложении. Продолжение использования приложения после публикации изменений означает ваше согласие с ними.
